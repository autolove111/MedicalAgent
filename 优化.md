### **核心方法 3：process_query (L566-577)**

```python
def process_query(self, query, user_context=None, lab_results=None):
    """同步查询（一次性返回）"""
    messages, sources, _ = self._build_messages(query, user_context, lab_results)
    response = get_llm(streaming=False).invoke(messages)  # 等待完整响应
    content = response.content
    cleaned, metadata = extract_metadata(content)  # 提取元数据
    return cleaned, sources
```

### **核心方法 4：stream_query (L579-604) ⭐**

```python
def stream_query(self, query, user_context=None, lab_results=None) -> Iterator[Dict]:
    """流式查询（实时推流）"""
    messages, sources, _ = self._build_messages(query, user_context, lab_results)

    full_text = ""
    for chunk in get_llm(streaming=True).stream(messages):  # 流式调用
        text = chunk.content
        if text:
            full_text += text
            yield {"type": "delta", "content": text}  # ← 逐块发送给前端！

    # 最后发送元数据
    _, metadata = extract_metadata(full_text)
    yield {
        "type": "meta",
        "metadata": metadata,
        "sources": formatted_sources,
    }
```

**这就是 SSE 流式推送的实现！**
